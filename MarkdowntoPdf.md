# This is a PDF file

## It was converted from markdown to a pdf using:

* Pandoc
* Beamer ( LaTex )
* Texlive ( Most dependencies installed using yay )

# How to convert a markdown file to pdf?

## Command usage:

`pandoc pathtoyourfile.md -t beamer -o yourfile.pdf`
